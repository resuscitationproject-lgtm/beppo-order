# iPad運用手順

## 初回

1. iPad Safariで `admin.html` を開く。
2. 共有 → ホーム画面に追加 → 名前を「Beppo管理」にする。
3. ホーム画面から起動し、PINでログイン。
4. 通知音をONにする。

## 毎朝

提供予定数、事前取り置き数、本日のメニュー・説明・店主コメント・ビーガン可否を確認し、「営業開始」をタップします。接続表示と最終同期時刻も確認します。

## 営業中

- 店頭販売ごとに「店頭販売 −1食」。誤操作は5秒以内にUndo。
- 注文カードを受取順に確認し、受付済み → 調理中 → 受取待ち → 受取済みへ進める。
- 残り3食以下では10秒ごと、それ以外は15秒ごとに自動更新。
- オフライン中は表示だけを参照し、操作しない。復帰後に自動同期される。

## 閉店

未受取注文を確認して「営業終了」。売り切れ時は「売り切れ」。臨時休業は「休業」を使います。画面は長時間表示するため、充電と端末の自動ロック方針を店舗で決めてください。
